# ai-framework

Репа-мать персонального фреймворка работы с AI-агентами: скиллы, методология, рыбы проектных файлов. Источник правды — здесь; рабочие копии раскатываются наружу. Модель синхронизации и layout — в [README.md](README.md).

## Правила работы в этой репе

- Репа — источник правды для скиллов. `/skill-sync` приводит `skills/<name>/SKILL.md`, `~/.claude/skills/` и `~/.codex/skills/` к байт-в-байт одинаковому состоянию и запускается только по явной команде пользователя, никогда автоматически после правки.
- `CLAUDE.md` и `AGENTS.md` — и корневые, и рыбы в `templates/` — правятся парой, содержимое идентично.
- `_methodology.md` — источник правды методологии; проектные копии (`docs/_methodology.md`, gitignored в проектах) раскатываются отсюда, правки из проектов доносятся сюда.
- Файлы в `templates/` — рыбы для нового проекта: копируются и заполняются, здесь остаются с плейсхолдерами.
- Ветка `main`, remote `origin` → github `Kromeshn/ai-framework`.
